How to Raise a Ticket
Overview Step-by-Step Guide FAQ

Overview

Raising a ticket is the quickest way to get assistance and ensure your issue is tracked and resolved promptly.

Step-by-Step Guide

  1. Access the ticketing system: Visit the Help Center and log in if prompted.
  2. Select Request Type: Choose "IT SUPPORT (NOC)".
  3. Fill out ticket details:
    • Topic: Select the specific topic related to your request.

      • Ticket Details

      Category Description
      Access & Permissions Account management, onboarding/offboarding, password resets.
      Asset Management Asset disposal, relocation, and information requests.
      Email Issues Aliases, auto-replies, mailing lists, and troubleshooting.
      Google Drive/Email Upgrades Storage upgrades, file recovery, and ownership changes.
      Hardware Issues Device repair, printer/scanner support, and replacements.
      Network Issues Connectivity problems, slow performance, or outages.
      Software Requests Installation, upgrades, troubleshooting.
      Telephony Issues VoIP/Xcally problems, modifications, and call recording retrieval.
    • Country: Select "Kenya" to route to the correct local IT team.
    • Subject: Write a concise title (e.g., "Password Reset Request").
    • Description: Clearly explain the issue or request, providing relevant details such as affected systems or steps to reproduce the problem.(e.g.,.
      Requesting password reset for the agent below J*******j***.ext@jumia.com(input agent’s email)
      Justification:Agent just returned from contract break.
      +E****j***@jumia.com (input manager’s email) for approval.)
    • Attachments:You can upload any supporting files & screenshots for clarity.
  4. Submit your ticket: Review details and click "Submit".
  5. Track your ticket: Check your email for the confirmation message and ticket ID.

FAQ

What if my issue is urgent? Mark it as high priority when submitting.

How do I check my ticket status? Log into the Help Center and go to 'My Tickets'.
